## Installation
### JUCE5
Add this repository to your `Header Seatch Path` on Projucer.

### JUCE6
Add this repository to the `target_include_directories` in CmakeLists.txt.
```
target_include_directories(PROJECT_NAME PRIVATE "path/to/jx")
```
